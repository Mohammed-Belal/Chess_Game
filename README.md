ChessHub Realtime: 
link: https://chesshub-realtime.onrender.com
A functional, real-time multiplayer chess application that allows two players to compete in synchronized rooms while others join as spectators.

🚀 Features
 Real-Time Play: Instant move synchronization using Socket.io.

 Move Validation: Enforces official chess rules via the Chess.js engine.

 Spectator Mode: Allows additional users to watch ongoing games in real-time.

 Dynamic UI: Interactive board with drag-and-drop functionality.

🛠️ Tech Stack
Backend: Node.js, Express.js

Real-Time: Socket.io

Chess Logic: Chess.js

Frontend: JavaScript, EJS, Tailwind CSS

📋 Prerequisites
Node.js installed on your machine.

⚙️ Installation & Setup
Clone the repository:

Bash
git clone <your-repository-url>
Install dependencies:

Bash
npm install
Start the server:

Bash
node app.js
Open in browser: Navigate to http://localhost:3000
